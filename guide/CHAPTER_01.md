# 第1章：準備の儀式（開発環境の構築）

ウェブサイトを「つくる」ために、まずは自分だけの道具をパソコンの中に揃えましょう。
これから、「プログラミング」という魔法を使うための「工房（開発環境）」を整えます。

---

## 🏗️ 必要な3つの道具

### 1. Node.js（ノード・ジェーエス）
ウェブサイトを動かすための「エンジン」です。これが無いと何も始まりません。

*   **何をするもの？**: あなたが書いたプログラムを、パソコンが理解できる言葉にして実行してくれるものです。
*   **インストール方法**: [Node.js 公式サイト](https://nodejs.org/) から「LTS（推奨版）」を選んでダウンロードし、実行してください。全てデフォルト設定で進めてOKです。

### 2. VS Code（ブイエスコード）
「魔法の書（コード）」を書くための、高機能なメモ帳です。

*   **何をするもの？**: 正式名称は Visual Studio Code。世界中のエンジニアが使っている、最も人気のある「コードエディタ」です。
*   **インストール方法**: [VS Code 公式サイト](https://code.visualstudio.com/) からダウンロードし、インストールしてください。
*   **日本語化**: 起動後、左側のアイコン（四角いブロックのようなマーク）をクリックし、`Japanese Language Pack` と検索してインストールしてください。

### 3. ターミナル（黒い画面）
パソコンに「直接命令を出す」ための画面です。

*   **何をするもの？**: マウスでポチポチするのではなく、キーボードで文字を打って「プログラムを動かせ！」と命令します。
*   **使い方**: Macなら「ターミナル」、Windowsなら「コマンドプロンプト」や「PowerShell」という名前で最初から入っています。

---

## 🛠️ 最初のプロジェクトを作る

道具が揃ったら、いよいよ自分のプロジェクトを作ります。ターミナル（さっきの黒い画面）を開いて、以下の魔法の言葉（コマンド）を打ち込んでください。

```bash
npx create-next-app@latest my-archive
```

### この一行にはどんな意味があるの？（全行解説）

1.  **`npx`**: インターネットから特定のツールを「一度だけ借りてきて実行する」という命令です。
2.  **`create-next-app`**: 「Next.js」というウェブサイトの雛形（テンプレート）を新しく作るためのツールの名前です。
3.  **`@latest`**: 「最新のバージョンを使ってね」という意味です。
4.  **`my-archive`**: これから作るフォルダの名前です。自分の好きな名前にしても構いません。

---

## ❓ 途中で聞かれる質問への回答

コマンドを打つと、いくつか質問が出てきます。初心者の方は、**とりあえず全て「Enterキー」**を押して進めて大丈夫です。

1.  **TypeScript?**: はい（Yes）。より安全にコードを書くための仕組みです。
2.  **ESLint?**: はい（Yes）。コードの書き間違いを教えてくれる機能です。
3.  **Tailwind CSS?**: はい（Yes）。デザインを楽にするための道具です。
4.  **`src/` directory?**: はい（Yes）。コードを整理整頓して置くためのフォルダです。
5.  **App Router?**: はい（Yes）。最新の「ページ切り替え」の仕組みです。
6.  **Customize the default import alias?**: いいえ（No）。そのままの名前で使います。

---

## 🚀 最初のウェブサイトを表示してみる！

インストールが終わったら、作ったフォルダの中に移動して、サイトを起動してみましょう。

```bash
cd my-archive
npm run dev
```

### この二行にはどんな意味があるの？（全行解説）

1.  **`cd my-archive`**: 「Change Directory」の略。さっき作った `my-archive` というフォルダに「入る」という意味です。
2.  **`npm run dev`**: 「開発モード（development mode）で自分だけのローカルサーバーを起動せよ」という命令です。

実行して、ターミナルに `http://localhost:3000` と表示されたら、ブラウザでそのアドレスを開いてみてください。
**「Next.js のロゴ」が表示されましたか？それが、あなたのウェブサイト制作の第一歩です！**
