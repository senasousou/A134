# 第6章：美しさを宿す（Tailwind CSS とデザイン）

アーカイブの価値は、その「佇まい」で決まります。
ただ文字が並んでいるだけではなく、まるで博物館の展示ケースを覗いているような、静謐で学術的な空気感をどう作るか。
この章では、**Tailwind CSS** を使ったデザインの秘訣を学びます。

---

## 🎨 色の設計図（src/app/globals.css）

このプロジェクトの基調となる色は、**「Cosmic Latte（宇宙のラテ）」** と呼ばれる、宇宙全体の平均的な色に近いベージュです。

### 全行解説

```css
1: @import "tailwindcss";             // Tailwind CSS というデザインの道具箱を読み込みます
3: @layer base {                      // 基礎となる設定を書く場所
4:   :root {                          // サイト全体で使い回す「色の変数」を定義
5:     --background: #FFF8E7; /* 宇宙の色 */ // 背景色：目に優しいベージュ
6:     --foreground: #1c1917; /* インクの色 */ // 文字色：真っ黒ではない、深い石のような黒
8:     --muted: #f5ebd5;               // 少し控えめな背景色（カードなど）
11:     --border: #e7e0cd;             // 繊細な区切り線の色
12:   }
13: }
16: body {                            // サイト全体の文字の基本ルール
17:   background-color: var(--background); // 5行目で決めた背景色を使う
18:   color: var(--foreground);            // 6行目で決めた文字色を使う
19:   font-family: 'Helvetica Neue', ...;  // 読みやすいゴシック体のフォントセット
21:   letter-spacing: 0.02em;              // 文字の間隔をわずかに広げて「高級感」を出す
22: }
24: h1, h2, h3, h4, h5, h6 {          // 見出し（タイトル）の設定
25:   font-family: 'Yu Mincho', ...;       // 日本語の美しい「明朝体」を使う
26:   font-weight: 500;                    // 太すぎず細すぎない、上品な太さ
27: }
```

---

## 📐 デザインの3大原則：Archive134 スタイル

初心者がデザインをするときに意識すべき、このプロジェクトの「ルール」が3つあります。

### 1. 「モノスペース（等幅フォント）」をアクセントに
数字や記号（資料IDなど）には、プログラミングコードのような **`font-mono`** クラスを使います。
「機械的に管理されている」という理知的な印象を与えることができます。

### 2. 「ボーダー（線）」で情報を区切る
派手な色ではなく、細い薄い線（`border-[#bbb4a4]/30`）を引くことで、静かな整理整頓を表現します。

### 3. 「セリフ（明朝体）」で重みを出す
タイトルや長い文章には **`font-serif`** を使います。
画面が「ウェブページ」から「紙の記録資料」へと変わる魔法です。

---

## ⚡ Tailwind CSS の便利な書き方

例えば、HTMLの中にこう書きます：
`<div className="mb-4 text-center border-b p-2">`

*   **`mb-4`**: Margin Bottom。下の余白を「4」の広さ分あける。
*   **`text-center`**: 文字を真ん中に寄せる。
*   **`border-b`**: Border Bottom。下に細い線を引く。
*   **`p-2`**: Padding。内側の余白を「2」の広さ分あける。

このように、**「英単語の略称」** をクラス名として書くだけで、CSSを一行も書かずにデザインができるのが、Tailwind CSS の凄さです。
