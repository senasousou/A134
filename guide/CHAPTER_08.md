## 🏗️ 公開（デプロイ）の仕組み

これまではあなたのパソコンの中だけでサイトを動かしていましたが、**Vercel（バーセル）** というサービスを使うことで、あなたのコードを「インターネット上の専用サーバー」に預け、24時間いつでもアクセスできるようにします。

---

## 🛠️ Vercel への公開手順（手取り足取り）

### 1. GitHub にコードを預ける
まずは自分のコードを **GitHub** という大きな倉庫にアップロードします。

*   GitHub で「New Repository」を作成します。
*   自分のパソコンから、ターミナルで以下の魔法（コマンド）を使ってコードを送ります。
    ```bash
    git add .
    git commit -m "初のアーカイブ完成！"
    git push origin main
    ```

### 2. Vercel と連携する
[Vercel 公式サイト](https://vercel.com/) にアクセスし、GitHub アカウントでログインします。

*   **[Add New]** → **[Project]** をクリックします。
*   さっき作った GitHub のリポジトリを **[Import]** します。
*   **[Deploy]** ボタンを押します。

**これで、数分後には `xxxx.vercel.app` というアドレスであなたのサイトが公開されます！**

## 🌐 独自ドメインを繋げる（おまけ）

今回のプロジェクトで設定した `senasousou.com` のように、自分だけの特別な名前をサイトに付けたい場合の手順です。

1.  **ムームードメインなどでドメインを買う**: 好きな名前を手に入れます。
2.  **Vercel の設定画面へ**: 「Settings > Domains」で、買った名前（例: `senasousou.com`）を入力します。
3.  **DNSレコードの修正**: Vercel から指定された数字（IPアドレス）を、ムームードメイン側の設定画面に書き込みます。

> [!TIP]
> **「ドメインの設定が反映されない！」** と思ったら、ムームーDNSの「設定1」で古いサーバー設定が残っていないか確認してみてください。第6章のデザインと同じく、不要なものを「削ぎ落とす」のがコツです。
